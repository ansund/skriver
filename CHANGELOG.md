# Changelog

All notable changes to `skriver` will be documented in this file.

The format is inspired by Keep a Changelog and the project follows Semantic Versioning.

## [0.1.0] - 2026-03-25

### Added

- local-first CLI transcription workflow for audio and video
- OCR-assisted video context capture
- optional local pyannote diarization
- summary draft and low-confidence artifact generation
- unit and mocked end-to-end tests
- open-source repo scaffolding for the initial public refactor
