# Support

Use the right channel so questions and bugs are easier to handle.

## Use issues for

- reproducible bugs
- install failures
- feature requests
- regression reports

## Use discussions for

- workflow questions
- agent integration ideas
- usage examples
- roadmap feedback

## Include when asking for help

- platform and OS version
- Node and Python versions
- `pnpm test` result
- `node src/cli.mjs transcribe --help` output if relevant
- the exact command you ran
- any relevant files or logs
